CalendarMaker

An easy way to work with calendars.

Months

To create the current month. 

CalendarMaker.month("#thismonth");

Years

To create the current year. 

CalendarMaker.year("#thisyear");

Weeks

To create the current week. 

CalendarMaker.week("#thisweek");

Agenda Views

To create an agenda view. 

CalendarMaker.agenda("#thisagenda");
